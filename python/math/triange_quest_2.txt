----------------

1:     1     =     1
2:    121    =    111    +    10
3:   12321   =   11111   +   1210
4:  1234321  =  1111111  +  123210
5: 123454321 = 111111111 + 12343210
-----------------------------------
therefore, 
  nth-term = (2n-1)th-term(111...) + (n-1)th-term x 10

again, 
1:     1     =     1
2:    121    =    111    +                             (1)x10
3:   12321   =   11111   +                      (111 + (1)x10)x10
4:  1234321  =  1111111  +             (11111 + (111 + (1)x10)x10)x10
5: 123454321 = 111111111 + (1111111  + (11111 + (111 + (1)x10)x10)x10)x10

5:           = 111111111      + 1111111x10   + 11111x10x10 +  111x10x10x10 +    1x10x10x10x10
5:           = 111111111x10^0 + 1111111x10^1 + 11111x10^2  +  111x10^3     +    1x10^4
5:           =      f(9)x10^0 +    f(7)x10^1 +  f(5)x10^2  + f(3)x10^3     + f(1)x10^4

different approach,
1:     1     =                                                                     1
2:    121    = 100                                                          + 20 + 1
3:   12321   = 10000     + 2000                                       + 300 + 20 + 1
4:  1234321  = 1000000   + 200000   + 30000                    + 4000 + 300 + 20 + 1
5: 123454321 = 100000000 + 20000000 + 3000000 + 400000 + 50000 + 4000 + 300 + 20 + 1
